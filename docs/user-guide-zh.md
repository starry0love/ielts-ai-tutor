# 雅思 AI 私教新手使用攻略

这份攻略写给第一次看到这个项目的人：不懂编程也没关系，你只需要知道它是做什么的、适合谁、怎么开始用。

## 这是什么

雅思 AI 私教是一个本地运行的 IELTS 学习导师。它不是题库软件，也不是官方评分器。它更像一个每天陪你做计划、盯复盘、看写作问题的学习助手。

你把自己的目标、基础、每天可用时间告诉它；它会先理解你的情况，再帮你安排今天该学什么。每天学完后，你再逐条复盘，它会根据你的真实执行情况调整后面的计划。

## 它适合谁

适合这些人：

- 想备考 IELTS，但不知道每天该学什么。
- 做了很多计划，但经常坚持不下去。
- 写作没人反馈，不知道自己为什么卡在 6 或 6.5。
- 不想把学习数据放到陌生云端，更希望本地保存。
- 有自己的大模型 API Key，想用自己的模型服务。

不太适合这些人：

- 只想找一套现成题库刷题。
- 想要官方 IELTS 分数保证。
- 完全不想配置任何大模型接口。

## 它能做什么

### 1. 连接你自己的大模型

第一次打开时，它会先要求你填写：

- 接口地址
- 模型名称
- API Key

这些信息只保存在你的电脑上。开源版不会内置作者的 API Key。

### 2. 建立你的学习画像

你需要告诉导师：

- 目标分数是多少
- 什么时候考试
- 每天大概能学多久
- 现在基础如何
- 以前怎么学
- 哪些方法有效，哪些方法无效
- 希望导师严格一点、温和一点，还是更像朋友

这一步很重要。它决定后续计划是不是贴合你，而不是泛泛地说“背单词、练听力、写作文”。

### 3. 生成今日计划

它不会直接扔给你一堆任务，而是先问今天的真实状态：

- 今天有多少时间
- 精力怎么样
- 想轻一点还是挑战一点
- 有没有特殊阻碍

然后再生成今天的学习任务。

### 4. 写作批改

你可以把自己的 Task 1 或 Task 2 作文贴进去。AI 会给出：

- 预估 band
- 四项评分维度反馈
- 主要问题
- 修改建议
- 下一步训练重点

注意：这是 AI 预估，不是官方 IELTS 分数。

### 5. 每日复盘

每天计划里有几条任务，你就复盘几条任务。复盘内容包括：

- 是否完成
- 实际用了多久
- 难度如何
- 质量如何
- 卡在哪里
- 明天怎么调整

这个项目的核心不是“生成计划”，而是“计划 -> 执行 -> 复盘 -> 调整”的闭环。

### 6. 学习进度和周/月报

用几天之后，它会开始积累：

- 学习时间
- 完成率
- 任务质量
- 写作反馈
- 低信号维度
- 常见阻碍

周报和月报会帮你看出：你到底是不会学、没时间、精力不稳，还是写作/口语/阅读某个维度一直拖后腿。

## 怎么启动

如果你拿到的是源码版本：

```powershell
cd C:\path\to\github-Ielts
powershell -ExecutionPolicy Bypass -File .\scripts\start-app.ps1
```

如果桌面窗口没有弹出，可以打开浏览器：

```text
http://127.0.0.1:5173
```

## 怎么填写大模型接口

你需要一个兼容 OpenAI 格式的接口。常见字段是：

```text
接口地址：https://api.example.com/v1
模型名称：你的模型名
API Key：你的 key
```

如果不确定接口地址和模型名，请看你所使用模型服务商的文档。只要它支持 OpenAI-compatible Chat Completions，一般就可以使用。

保存前建议先点“测试连接”。测试通过后再保存。

## 推荐的第一天用法

第一天不要急着追求完整计划，建议这样用：

1. 先连接大模型。
2. 完成入门画像，尽量写真实一点。
3. 让导师生成今天计划。
4. 只完成一个最核心任务。
5. 写一条认真复盘。

如果你有作文，第一天也可以贴一篇进去，让导师先建立写作反馈信号。

## 常见问题

### 它会不会泄露我的 API Key？

API Key 保存在你的本机配置里，不会提交到 GitHub，也不会内置在开源版安装包里。但你使用第三方大模型时，模型服务商会收到你发送给它的内容。

### 它有没有内置题库？

没有。开源版不内置第三方教材、音频、OCR 文本或官方练习内容。

### 它给的分数准吗？

它给的是 AI 预估 band，不能当成官方分数。更重要的是看它指出的问题和下一步建议。

### 我不会编程能用吗？

如果别人已经给你打包好便携版，你基本只需要双击打开、填写大模型接口。如果你拿到的是源码版，第一次启动会需要安装 Node.js 和 Python 依赖。

### 为什么不是安装包？

开源项目通常更推荐先提供源码和免安装便携版。这样用户更容易检查、删除、移动，也不需要写入系统安装目录。正式安装器以后可以作为可选发布方式。

## 一句话总结

IELTS AI Tutor 不是一个“题库”，而是一个“本地 AI 学习导师”。它最适合帮你把每天的学习变成可执行、可复盘、可调整的闭环。
